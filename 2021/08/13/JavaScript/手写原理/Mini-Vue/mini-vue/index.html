# Mini-Vue 实现思路

  • 渲染系统，改模块主要包含三个功能：
    • 功能一：h 函数，用于返回一个 VNode 对象；
    • 功能二：mount 函数，用于将 VNode 挂载到 DOM 上；
    • 功能三：patch 函数，用于对比两个 VNode 进行对比，决定如何处理新的 VNode；
  • 响应式系统：
    • 通知依赖 notify ()
    • 数据劫持 reactive ()
    • 数据结构 getDep ()
    • 添加方法 depend ()
    • 方法依赖 watchEffect ()
    • Vue2 的 defineProperty 实现
    • Vue3 的 Proxy 实现
