# 插槽

  1. 作用：让父组件可以向子组件指定位置插入 html 结构，也是一种组件间通信的方式，适用于 <strong style="color:red"> 父组件 ===> 子组件 </strong> 。
  2. 分类：默认插槽、具名插槽、作用域插槽
